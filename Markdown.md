## Teil 1: Die Reparatur-Werkstatt
*Das Markdown hier unten ist "kaputt" und wird in der Vorschau nicht richtig angezeigt. Deine Aufgabe ist es, die Syntaxfehler zu beheben, damit es korrekt aussieht.*

### 1. Korrigiere die Überschriften
#Dies sollte eine Überschrift 1 sein (hier fehlt ein Leerzeichen)
##Dies sollte eine Überschrift 2 sein
###Dies sollte eine Überschrift 3 sein

### 2. Korrigiere den Text-Stil
Dieser Satz sollte **fett** sein, aber die Syntax ist unvollständig.
Dieser Satz sollte _kursiv_ sein, aber es sieht falsch aus.
Dieser Text sollte ~~durchgestrichen~~ sein, aber die Tilden fehlen.

### 3. Korrigiere die Listen
* Dies ist ein Aufzählungspunkt (hier fehlt ein Leerzeichen)
* Dies ist ein weiterer Punkt
1.Dies ist eine nummerierte Liste (hier fehlt ein Leerzeichen)
2.Dies ist der zweite Punkt

  ### 4. Korrigiere den Link
Hier ist ein Link zu [Google] (https://google.com) (entferne das Leerzeichen zwischen den eckigen und runden Klammern).
